lab_3.c, lab_3_staticLib.c и lab_3_dynamicLib.c - это файлы из 3 РАЗНЫХ проектов.

Для работы лабы со статической библиотекой надо:
1) скомпилировать lab_3_staticLib.c
2) во вкладке Workspace (слева, где показаны все файлы) правой кнопкой мыши нажимаем на основной проект, и выбираем "Activate Project"
3) В панели вверху нажимаем Project -> Build Options -> Linker Settings -> Add. 
4) Там ищем проект библитеки, заходим в папку bin -> debug -> выбираем библиотеку (файл с форматом 'a')
5) Компилируем lab_3.c
6) В проводнике ищем exe файл от lab_3.c (bin -> debug -> *тут exe*), но не запускаем его
7) Находясь в проводнике, нажимаем на путь папки, и меняем выделившийся текст на 'cmd'
8) В открывшейся консоли можно запускать лабу, например введя 'lab_3.exe Lebovski 10, 5, -20' (результатом будет -5)
