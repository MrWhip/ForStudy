lab_3.c, lab_3_dllVersion.c, lab_3_staticLib.c и lab_3_dynamicLib.cpp - это файлы из 4 РАЗНЫХ проектов.

Примечание: файл lab_3.c - это файл main.c в проекте lab_3. То есть вам надо создать проект со своим названием, и этим кодом заменить свои main файлы.
main.c/main.h/main.cpp советую не переименовывать, просто замените содержимое. В итоге должно получиться 4 проекта со своими main файлами.

Для работы лабы со статической библиотекой надо:
1) скомпилировать lab_3_staticLib.c
2) во вкладке Workspace (слева, где показаны все файлы) правой кнопкой мыши нажимаем на основной проект, и выбираем "Activate Project"
3) В панели вверху нажимаем Project -> Build Options -> Linker Settings -> Add. 
4) Там ищем проект библитеки, заходим в папку bin -> debug -> выбираем библиотеку (файл с форматом 'a')
5) Компилируем lab_3.c
6) В проводнике ищем exe файл от lab_3.c (bin -> debug -> *тут exe*), но не запускаем его
7) Находясь в проводнике, нажимаем на путь папки, и меняем выделившийся текст на 'cmd'
8) В открывшейся консоли можно запускать лабу, например введя 'lab_3.exe Lebovski 10, 5, -20' (результатом будет -5)

Для работы лабы с динамической библиотекой надо:
1) скомпилировать саму библиотеку
2) скомпилировать основу лабы
3) в проводнике ищем dll по адресу *библиотека*->bin->debug и копируем этот dll-файл в *проект*->bin->debug к основному exe-файлу
4) запускаем exe-файл так же, как показано в шагах 6-8 для статической библиотеки
